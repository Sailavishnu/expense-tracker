# Expense Tracker

A responsive web-based expense tracking application built as part of my web development learning journey.

## About the Project
Expense Tracker helps users record and manage their daily expenses in a simple and organized way.  
The application allows users to add, view, and delete expenses, making it easier to understand spending patterns.

This project focuses on applying core web development concepts while building a practical, real-world use case with a clean and user-friendly interface.

## Features
- Add and delete expenses
- Dynamically update the expense list
- Visual representation of expenses using charts
- Simple and intuitive user interface
- Responsive design for both mobile and desktop devices
- Data persistence using browser localStorage

## What I Learned
- Structuring a real-world web application
- Handling user input and DOM events
- Managing and visualizing data using JavaScript
- Using charts to represent expense data clearly
- Improving layout, usability, and user flow
- Working with browser localStorage for data persistence
